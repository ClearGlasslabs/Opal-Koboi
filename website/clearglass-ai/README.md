# ClearGlass.ai Website Bundle

Futuristic multi-page static website bundle for ClearGlass.ai.

## Included

- Glassmorphism UI
- Dark/light theme toggle
- Reduced motion toggle
- SEO + Open Graph metadata
- Static deploy ready
- Mobile responsive navigation

## Deploy

### GitHub Pages

1. Push repository
2. Settings → Pages
3. Deploy from `/website/clearglass-ai`

### Netlify / Vercel / Cloudflare Pages

Deploy the `website/clearglass-ai` directory as a static site.
